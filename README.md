# 🍽️ RestaurantePOS v2.0

Sistema de punto de venta para restaurantes.
Desarrollado con Python + Flet + SQLite.

## Módulos
- ✅ Dashboard con estadísticas en tiempo real
- ✅ Nueva Venta con carrito y cobro
- ✅ Productos CRUD con imágenes
- ✅ Reportes con historial y PDF
- ✅ Caja con cortes diarios
- ✅ Clientes CRUD con historial
- ⏳ Configuración
- ⏳ Login/Auth
- ⏳ Android APK

## Instalación
```bash
pip install -r requirements.txt
python main.py
```

## Stack
- Python 3.14
- Flet 0.21.2
- SQLite (embebida)
- fpdf2
- matplotlib
